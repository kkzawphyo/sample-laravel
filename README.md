Simple Laravel Blog Website
===========================

Overview
--------
This project is a simple web application built with the Laravel framework.
It provides basic user authentication and allows users to create posts and
add comments.

Features
--------
- User registration
- User login and logout
- Authentication and session management
- Create, edit, and delete posts
- Comment on posts
- Access control for authenticated users

Technology Stack
----------------
- Backend: Laravel
- Frontend: Blade Template Engine
- Database: MySQL
